Muninn CLI reference

Pi feeder flow

readsb writes /run/readsb/aircraft.json → upload-if-ready.sh (skips when no positioned aircraft) → muninn.py --upload → wdgwars.pl

Go live (CLI)

./scripts/go-live.sh YOUR_API_KEY [minutes]

Useful commands

  • muninn.py --whoami — verify API key and show scores
  • muninn.py --save-key KEY — store API key
  • muninn.py aircraft.json --upload — one-shot upload
  • muninn.py file --upload --dry-run — preview without sending
  • muninn.py --config — show key status (not the key itself)

Input formats (auto-detect)

AVR raw Mode-S, SBS-1/MSG CSV, dump1090/readsb aircraft.json, generic CSV, BaseStation .sqb
